Model Management


Since the game characters share common properties, we can manage them via a single Object Manager. They will all have to be drawn, updated, and removed when they are no longer active.

ObjectManager

1. Create a new class called ObjectManager and add a rocket member variable
BUT DO NOT INITIALIZE IT


2. Make a constructor for this class that takes a Rocketship parameter and us it to initialize the member variable created above.


3. Add a member variable that will hold an ArrayList of Projectile objects.
Ass a member variable that will hold an ArrayList of Alien objects.


4. Create a method called update() that calls the rocket's update method. Then, iterate through all the aliens and call the update method for each. After updating, if the alien's x and y values place it outside the game window (use LeagueInvaders.WIDTH and LeagueInvaders.HEIGHT), set the alien's isAlive variable to false. Note: Iteration will require the use of a for loop of some kind! Now do the same thing with all the projectiles.


5. Create a method called draw() that has a Graphics parameter. Add code to call the rocket's draw method passing it the graphics object. Then iterate through all the aliens and call the draw method for each. Now do the same thing for all the projectiles.

GamePanel

6. Inside the GamePanel class, create and initialize a member varable for an ObjectManager object. Use the panel's rocketship as the parameter for the ObjectManager constructor.


7. In the updateGameState method, delete the call to the rocket's update method and replace it with a call to the objectManager's update method.

8. In the drawGameState method, delete the call to the rocket's draw method and replace it with a call to the objectManager's draw method.


TESTING

9. Run the program. You should see no change from the previous test. That is, you should still see a blue rocket when in the GAME state, and your background color should still change when you press the Enter key.
