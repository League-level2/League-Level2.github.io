Alien Timer


A Timer will control the rate at which enemies will appear in the game. The following steps will create and start the timer as part of the GamePanel function, but the aliens will be added as part of the ObjetManager function.

ObjectManager


The ObjectManager is going to use the ActionListener interface to add an alien to the game each time it expires. Make the ObjectManager class implement the ActionListener interface as shown below:

public class ObjectManager implements ActionListener
{ ... }

Add the unimplemented method (Eclipse can help). You should now have an actionPerformed() method in this class.


Each alien should have a random location across the top of the game, so in this method add the following:

addAlien(new Alien(new Random().nextInt(LeagueInvaders.WIDTH), 0, 50, 50));

GamePanel

1. In the GamePanel class add another Timer member variable called alienSpawn. You should already have imported javax.swing.Timer when you set up the main game timer.


2. In the GamePanel constructor, initialize the alienSpawn member variable. The constructor of the Timer class takes two parameters. The first parameter is an int for how fast your want the timer to repeat in milliseconds. Let's start by having one alien spawn every second. So the first parameter will be 1000. The second parameter takes a reference to an ActionListener object. We are going to implement the alienSpawn code in the ObjectManager, so we put objectManager as the second parameter.

    alienSpawn = new Timer(1000 , objectManager);

3. We only want the aliens to be created while the game is running, so create a new method startGame() to start the alienSpawn timer. This method must be called when the game state changes from MENU to GAME. Also add code to stop this timer when the game state changes from GAME to END.


TESTING

6. Run the program. Is the frame now filled by a blue rectangle. Is "action" appearing repeatedly on the console? If not, get this part working before you move on to the next step.
