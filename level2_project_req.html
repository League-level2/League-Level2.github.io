League Level 2 - Project Requirements

Project content:

     Game
          The Level 2 game project must consist of at least 3 Java classes.

     Slideshow
          A short presentation slideshow (5-10 minutes) must be produced that explains:

          Why you made this game
          Who is it designed for?
          What you learned from your usability testing
          How you adapted your project in light of user feedback
          Brief overview of the technical implementation

          Optional extras...

          The creative process for choosing the project
          Technical challenges along the way
          What you learned from the project
          How you worked as a team
          What you enjoyed
          How you could make it even better

Project testing:

     There must be at least one round of usability testing that includes a minimum of 3 test sessions recorded on usability testing log.

          Each test requires three people: The Tester, The Programmer (student), and The Moderator (teacher).
          The tester should be able to play the game without the programmer's help.
          The programmer can only give hints if the moderator approves.
          The programmer should take notes during the test.
          The tester should be interviewed at the end to find out how they enjoyed the game and if they have suggestions for improvements.
          The test session should be written up in a testing log and potentially used as part of the presentation detail.

testing log      Some of the feedback from the testing should feed into the design of the game.

Project delivery:

     Code must be published in student's public GitHub.
     Game must be packaged into a runnable JAR and demonstrated to work on presentation computer. How to
     Game must be uploaded to GitHub. How to
     Presentation slideshow must be presented to and approved by teacher.

To complete Level 2, the game project must be presented and demonstrated to a peer group and teachers.
Family members and friends may also attend the presentation event.
