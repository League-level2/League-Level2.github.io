Collisions

In order to make our objects react to colliding with each other, we will use Java's Rectangle class.

1. In your GameObject class, create an object of the Rectangle class in your member variables. Call this object collisionBox. You will have to import the class.

2. Initialize the collision box in the GameObject constructor. It takes four parameters. Use the x, y, width, and height variables.

3. In the update method of the GameObject class, update the collision box with the x, y, width, and height variables.

        collisionBox.setBounds(x, y, width, height);

4. Go to your Rocketship class. At the very top of the constructor, add this line:

        super();

5. At the very top of the update method inside the Rocketship class, add this line:

        super.update();

6. Add these same lines to the same methods in the Alien and the Projectile classes.

7. Go to the ObjectManager class and create a method called checkCollision.

8. Add code to the checkCollision method that checks if the player collides with any enemies. If so, set the player's isAlive variable to false.

for(Alien a : alienList){

        if(rocket.collisionBox.intersects(a.collisionBox){

                rocket.isAlive = false;

        }

}

9. Add code that iterates through all the projectiles and all the aliens. If any of them collide, set their isAlive variables to false.

10. create a method called purgeObjects. This method should use a standard for-loop to iterate through the aliens and projectiles. If any of them are not alive, then remove them from the list.

11. Go to the GamePanel class. In the updateGameState method, use your ObjectManager object to call the checkCollision method.

12. Run your program. Do the aliens and ship disappear when they are supposed to?

13. In the updateGameState method, create an if statement that checks the isAlive variable of your Rocketship object. If it is false, set the currentState equal to END_STATE
